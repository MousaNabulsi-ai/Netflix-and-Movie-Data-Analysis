# Netflix and Movie Data Analysis

This project focuses on cleaning, tidying, and merging two datasets (Netflix and Movies) using Python.
It also includes visualizations showing trends in movie popularity and production by country.

## Key Steps:
- Handling missing values and data types
- Removing duplicates
- Merging datasets
- Creating visual insights using matplotlib and pandas

## Tools Used:
- Python (Pandas, Matplotlib)
- Jupyter Notebook
